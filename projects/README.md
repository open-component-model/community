# Projects

This directory contains community-contributed OCM projects, each approved by SIG Runtime.

## Structure

Each project is an independent module in its own subdirectory. Every project MUST contain:

| File        | Required | Purpose                                   |
|-------------|----------|-------------------------------------------|
| `README.md` | Yes      | Describe what the project does and why    |
| `OWNERS.md` | Yes      | List project maintainers (GitHub handles) |

## Adding a Project

See [`CONTRIBUTING.md`](../CONTRIBUTING.md) for the full submission process.
